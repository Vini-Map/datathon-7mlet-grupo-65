# Synthetic enrichment — file schemas

> Generated by `aep synth report`. Artifacts (`*.parquet`, `*.json`) are
> git-ignored; regenerate with `aep synth generate`. Physically separate
> from the Kaggle base: these are an invented commercial/experimentation
> layer, grounded in the base only through context features and the real
> subscription signal.

## `offer_catalog.parquet` — the bandit arms

| Column | Type | Description |
|--------|------|-------------|
| `offer_id` | str | Arm identifier (O1..O8). |
| `name` | str | Human-readable offer name. |
| `product` | str | Product family (term_deposit, credit_card, ...). |
| `channel` | str | Delivery channel (cellular, telephone, app_push, email). |
| `message_tone` | str | Framing (informative, urgency, social_proof, reward). |
| `base_appeal` | float | Latent baseline conversion logit (pre-context). |
| `eligibility` | str | Human-readable suitability rule. |

## `offer_events.parquet` — logged impressions (one row per decision)

| Column | Type | Description |
|--------|------|-------------|
| `event_id` | int | Unique impression id. |
| `day` | int | Day of the impression in [0, horizon). |
| `client_idx` | int | Row index into the processed base (context reference). |
| `segment` | str | Synthetic fairness segment (age band x white-collar). |
| `subscribed` | int | Real Bank Marketing subscription signal (context feature). |
| `offer_id` | str | **Logged action**: the arm actually shown. |
| `n_eligible` | int | Number of eligible arms for this client. |
| `propensity` | float | P(action) under the logging policy = 1 / n_eligible. |
| `click` | int | Immediate (intermediate) reward: 1 if clicked. |

## `delayed_rewards.parquet` — final reward realization

| Column | Type | Description |
|--------|------|-------------|
| `event_id` | int | Joins to `offer_events`. |
| `conversion` | int | **Final reward**: 1 if converted (requires a click). |
| `reward_delay_days` | int | Days until the reward is observable. |
| `reward_available_day` | int | `day + delay` (converter) or `day + censor` (else). |
| `censored` | bool | True for non-converters (reward known to be 0 at horizon). |

## `reward_model_params.json` / `context_standardizer.json`

The frozen oracle reward parameters and the numeric-feature standardizer,
persisted for reproducibility and reuse by the bandits/service.
